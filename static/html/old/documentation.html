Documentation

API

The API can be accessed once you have signed up and retrieved an API key. CLICK HERE SIGN UP AND GET A KEY. The API key and your account email are used to create the constructor. API calls are then made to award your users. The functions include:
UserInfuser(account, api_key)
get_user_data(user_id)
update_user(user_id)
award_badge(user_id, badge_id)
remove_badge(user_id, badge_id)
award_points(user_id, points_awarded)
award_badge_points(user_id, badge_id, points_awarded, points_required)
get_widget(user_id, widget_type, height, width)
